## ランキング・戦績
- [] メイン画面の「設定」「クリエイト」ボタンの下に、「ランキング」「戦績」ボタンを追加。専用ページを追加。
- [] ランキング画面では、メモリーセットごとにランキングを閲覧できるようにする
- [] 戦績画面では、自分が解いてきた問題や最高成績、対戦の連続勝利数、苦手なキー、失敗した問題などを確認できるようにする(? 要件等)
  - [] 戦績に、自分の成長具合の分析を実装

## リザルト
- [] 暗記力を正答数ではなく正答率( = 正答数/出題数 )基準にする
- [] ゲーム終了条件によって、ランキングの評価基準を変更し、ランキングに登録するデータを変更する。
  - [] ゲーム終了条件が「正解数」の場合、「1問あたりの平均回答速度」を基準とする
  - [] ゲーム終了条件が「出題数」の場合、「正答率」を基準とする
  - [] 上記の変更に合わせて、ゲーム開始時の「...問クリア(正解)までのタイムを競おう！」の文言を変更

## メモリーセット
- [] 「問題セット」「メモリーセット」「Stage」「ガーデン」といった名称を統一する
- [] メモリーセットの公開設定を追加する
- [] 出題順序の「苦手優先」を正しく動くようにする

## 対戦
- [] 対戦相手が退出した場合に、そのゲーム終了まではそのまま進行し、再戦できなくする
- [] 対戦のゲーム終了後相手が退出していると、1人で再戦できてしまうバグを修正し、相手が退出すると「再戦」ボタンを無効にする
- [] ルーム作成時に、メモリーセットの設定によらず「終了条件」「問題数」を設定できてしまうのを、どちらかに統一する
  - [] 「終了条件」「問題数」をマルチプレイのルーム作成時や、ソロプレイのゲーム開始時に設定するように変更し、データセットの設定からは除外する
- [] 「ルーム一覧」からルームを削除できないようにする。削除は、ルームに一定時間誰もいない状態でのみ行う。
